

# ___Catalogue Service API___

##  Версии
* JDK 24
* Gradle 9.2.1
* SpringBoot 3.3.5

## Основной стек зависимостей
* Spring Security
* Spring Reactive Web
* Spring R2DBC
* Postgresql
* Kafka
* Reactive Redis
* JUnit

## Описание
    API интерфейс для организации работы с каталогами товаров

## Запуск приложения
* > Запустить приложение с помощью compose.yaml
  > Проверить запросы с помощью request-test.http
